Yii2 QR Code to create QR Code with yii2 framework.


Installation
-----------------

The preferred way to install this extension is through [composer](http://getcomposer.org/download/).

Either run

```
$ php composer.phar require cinghie/yii2-qrcode "*"
```

or add

```
"cinghie/yii2-qrcode": "*"
```
